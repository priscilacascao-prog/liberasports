Qual abordagem para a tela de Cadastrar Venda?

Clique na que combina mais com seu jeito de trabalhar

A

Layout 2 colunas

Esquerda: carrinho / produtos
Direita: cliente + entrega + pagamento

Tudo visível de uma vez só. Zero rolagem em telas grandes (desktop/notebook). Em celular vira uma coluna só.

B

Wizard em 3 etapas

1. Cliente & Produto → 2. Entrega → 3. Pagamento

Uma etapa de cada vez, com botão "Próximo". Menos informação na tela, mais clara, mas precisa clicar para avançar.

C

Seções colapsáveis

Tudo em uma coluna, mas cada seção abre/fecha. Você só vê a que está preenchendo.

Parecido com hoje, mas compacto. Não precisa clicar "próximo", só expandir o que interessa.
